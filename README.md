# `SECURE_VAULT` — Identity Creation UI

## 🔒 Description
`SECURE_VAULT` is a high-fidelity frontend prototype for generating secure node identities.  
The goal is to **demonstrate a decentralized key management UI** with a cyber-security aesthetic and smooth UX.

**Features:**
- Generate unique Node IDs
- Copy Node ID to clipboard
- Visual style with glass effects, scanning aura, and gradients
- Responsive layout for desktop and mobile
- Fully dark mode with Tailwind CSS
- Security-themed mockups (AES-256, P2P Mesh) for demonstration purposes

---

## ⚡ Technologies
- HTML5 + CSS3  
- TailwindCSS with custom theme and color palette  
- Google Fonts: Space Grotesk, Inter, Material Symbols  
- Vanilla JavaScript for ID generation and copy button
